Two Ways to Learn

Practice Mode vs Test Mode

Each mode is purpose-built — one for exploration and deep learning, the other for exam simulation and performance benchmarking.

Practice Mode
Learn at your pace
  • Unlimited time per question — no pressure
  • Instant answer reveal with explanation
  • Filter questions by topic or difficulty
  • Bookmark questions for later review
  • Skip and return to questions freely
  • Builds conceptual understanding
Test Mode
Exam simulation
  • Configurable countdown timer per session
  • Answers hidden until test completion
  • Custom question count (10, 25, 50…)
  • Detailed score report after submission
  • Track test history and score trends
  • Builds exam speed and confidence
